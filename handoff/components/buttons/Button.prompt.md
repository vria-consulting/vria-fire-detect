Bouton principal Kanari : pill arrondi, jaune canari par défaut — utiliser pour toute action ; `alert` (braise) réservé aux CTA d'urgence.

```jsx
<Button variant="primary" size="l">Protéger ma zone</Button>
<Button variant="dark">Voir la carte</Button>
<Button variant="ghost" size="s">En savoir plus</Button>
```

Variantes : primary (jaune/charbon), dark, ghost (bordure), alert (braise). Tailles s/m/l (36/44/52px). Hover assombrit le fond, press scale(0.98). Cible tactile ≥ 44px : préférer m/l sur mobile.
